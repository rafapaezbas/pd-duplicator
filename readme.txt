This is a script for duplication code of a pure data patch.

How to use this script?

duplicator.py --file --duplicationTimes --duplicationTokens

#program call -> duplicator.py C:/users/file 2 "select 98 99: key 99 100"

This will replace the instances of the object select with select 97, select 98 and the instances of key with key 99, key 100...
